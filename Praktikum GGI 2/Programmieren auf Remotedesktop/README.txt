Kleiner Hinweis zum Info 2 Praktikum:
Sollte jemand Probleme mit dem Download/der Installation/der Einrichtung von Visual Studio 2010 haben, lässt sich auch ganz bequem per Remotedesktopverbindung zu einem für euch virtuell von der RWTH bereitgestellten Arbeitsplatz unter Windows 7 programmieren.

0. Bevor das jedoch funktioniert, muss man sich im TIM diesen Dienst freischalten (einmalig!). siehe https://www.facebook.com/photo.php?fbid=4687392820455&set=o.249529785079892&type=1&theater
Das geht am schnellsten über diesen Link hier: https://tim.rz.rwth-aachen.de/enrole

1. Um die notwendige Verbindung zum RWTH-Netz aufzubauen (nur falls nicht sowieso schon drin), wird noch AnyConnect benötigt. Das gibts hier: http://www.rz.rwth-aachen.de/aw/cms/rz/Themen/unsere_dienste/kommunikation/netzbetrieb/dienste/vpn/~syt/Cisco_AnyConnect/?lang=de

2. Remotedesktopverbindung öffnen und mit dem Server (siehe Screenshot), Benutzernamen (ACHTUNG: "WIN-HPC\xx123456") und eurem vergebenen Kennwort anmelden.

3. Visual Studio ist bereits vorinstalliert -> öffnen.
